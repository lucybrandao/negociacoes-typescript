## Setup
Dentro da pasta do projeto, rodar o comando:

```shell
$ npm install
```

Depois, rodar o comando:

```shell
$ npm run start
```

O projeto abrirá uma janela no chrome automaticamente na porta 3000 (http://localhost:3000/).
